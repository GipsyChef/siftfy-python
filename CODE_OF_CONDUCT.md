# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

The official policy text is maintained by the Contributor Covenant project:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

Report unacceptable behavior privately to <hi@siftfy.io>. Maintainers may remove,
edit, or reject comments, commits, issues, and pull requests that do not align
with that policy.
